<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">YuLan-Chat模型是基于聊天的大语言模型，由中国人民大学GSAI的研究人员开发（YuLan，代表玉兰玉兰，是中国人民大学的校园花）。最新版本是通过从头开始进行预训练，并通过课程学习以高质量的英文和中文指令以及人类偏好数据进行监督微调而开发的。该型号具有以下技术特点：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">通过高质量的汉英双语数据的大规模预训练，模型的语言能力得到了提升。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">由于人类对齐的课程学习策略，我们的模型的有益性、诚实性和无害性得到了增强。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">为了更好地支持中文较长的输入和输出，我们用中文单词和最大输入长度来扩展词汇表。现在可以支持8k上下文了。</font></font></li>
</ul>
<blockquote>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">玉兰-聊天系列模型是中国人民大学高瓴人工智能学院师生共同开发的支持聊天的大语言模型（名字“玉兰”取自中国人民大学花）。最新版本从头完成了整个预训练过程，并采用基于中英文双语数据的课程学习技术进行有监督参数，包括高质量指令和人类偏好数据。该版本模型具有以下技术特点：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">由于在大规模中英双语数据上进行了继续预训练，模型的语言能力得到提高；</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">由于采用了课程学习方法进行人体扫描训练，模型在真实场景下的有用性、诚实性与无害性得到了增强；</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">为了更好的支持中文和更长的输入输出，模型的词表及长度得到了扩充，目前可支持8k上下文。</font></font></li>
</ul>
</blockquote>
